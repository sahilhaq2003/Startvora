### Color Palette

#### Primary Colors
- **Brand Primary**: #6366F1 (Indigo) - Main CTA buttons, links, active states
- **Brand Secondary**: #8B5CF6 (Purple) - Secondary actions, highlights
- **Success**: #10B981 (Green) - Positive actions, growth indicators
- **Warning**: #F59E0B (Amber) - Alerts, attention needed
- **Error**: #EF4444 (Red) - Errors, critical alerts
- **Info**: #3B82F6 (Blue) - Information, neutral notifications

#### Neutral Colors
- **Gray 50**: #F9FAFB - Backgrounds
- **Gray 100**: #F3F4F6 - Card backgrounds, subtle dividers
- **Gray 200**: #E5E7EB - Borders, inactive states
- **Gray 400**: #9CA3AF - Placeholder text, disabled text
- **Gray 600**: #4B5563 - Secondary text
- **Gray 900**: #111827 - Primary text, headings

#### Dashboard-Specific Colors
- **Revenue**: #10B981 (Green)
- **Expenses**: #EF4444 (Red)
- **Profit**: #6366F1 (Indigo)
- **Customer Growth**: #8B5CF6 (Purple)

### Typography

#### Font Family
- **Primary**: Inter (sans-serif) - UI, body text
- **Secondary**: Poppins - Headings, emphasis
- **Monospace**: Fira Code - Code snippets, data tables

#### Font Sizes
- **Heading 1**: 36px (2.25rem) - Page titles
- **Heading 2**: 30px (1.875rem) - Section titles
- **Heading 3**: 24px (1.5rem) - Card titles
- **Heading 4**: 20px (1.25rem) - Subsections
- **Body Large**: 18px (1.125rem) - Important text
- **Body**: 16px (1rem) - Standard text
- **Body Small**: 14px (0.875rem) - Secondary text
- **Caption**: 12px (0.75rem) - Labels, timestamps

#### Font Weights
- **Light**: 300 - Subtle text
- **Regular**: 400 - Body text
- **Medium**: 500 - Emphasis
- **Semibold**: 600 - Buttons, headings
- **Bold**: 700 - Strong emphasis

### Spacing System (8px base)
- **xs**: 4px
- **sm**: 8px
- **md**: 16px
- **lg**: 24px
- **xl**: 32px
- **2xl**: 48px
- **3xl**: 64px

### Border Radius
- **sm**: 4px - Buttons, inputs
- **md**: 8px - Cards, modals
- **lg**: 12px - Large cards
- **xl**: 16px - Hero sections
- **full**: 9999px - Pills, avatars

### Shadows
- **sm**: 0 1px 2px rgba(0,0,0,0.05) - Subtle elevation
- **md**: 0 4px 6px rgba(0,0,0,0.07) - Cards
- **lg**: 0 10px 15px rgba(0,0,0,0.1) - Modals, dropdowns
- **xl**: 0 20px 25px rgba(0,0,0,0.15) - Important overlays

**Features:**
- Contextual suggestions
- Action buttons in responses
- Can pull up relevant data/charts
- Voice input option
- Save conversations
- Available from any page (floating button)
- Learns from business data

---

## Responsive Design Breakpoints

### Desktop (1280px+)
- Full sidebar navigation
- Multi-column layouts
- Expanded charts and widgets

### Tablet (768px - 1279px)
- Collapsible sidebar
- 2-column grid
- Touch-optimized controls

### Mobile (< 768px)
- Hamburger menu
- Single column
- Bottom navigation bar
- Larger touch targets (44px minimum)
- Swipe gestures

---

## UI Components Library

### Buttons
- **Primary**: Indigo background, white text
- **Secondary**: White background, indigo border
- **Destructive**: Red background, white text
- **Ghost**: Transparent, gray text
- **Icon**: Square, icon only

### Form Inputs
- **Text Input**: Border, rounded corners, focus state
- **Select Dropdown**: Chevron icon, searchable
- **Date Picker**: Calendar popup
- **Toggle Switch**: iOS-style
- **Checkbox/Radio**: Custom styled

### Cards
- White background
- Shadow on hover
- Rounded corners (8px)
- Padding: 16-24px
- Optional header with actions

### Modals
- Overlay (rgba(0,0,0,0.5))
- Centered, max-width 600px
- Close button (top right)
- Slide-in animation

### Toast Notifications
- Top-right positioning
- Auto-dismiss (5 seconds)
- Color-coded by type:
  - Success: Green
  - Error: Red
  - Warning: Amber
  - Info: Blue

### Loading States
- Skeleton screens for content loading
- Spinner for actions
- Progress bar for uploads
- Shimmer effect for placeholders

---

## Accessibility (WCAG 2.1 AA)

- **Color Contrast**: 4.5:1 for normal text, 3:1 for large text
- **Keyboard Navigation**: Tab order, focus indicators
- **Screen Reader**: ARIA labels, semantic HTML
- **Alt Text**: All images and icons
- **Form Labels**: Clear, associated labels
- **Error Messages**: Descriptive, accessible
- **Motion**: Respect prefers-reduced-motion
